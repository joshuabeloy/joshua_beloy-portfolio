This is Me

The main goal of the study is to create software applications that can identify milk fish in live feed frames, estimate their size and weight using the YOLO algorithm, illustrate the data through graphs using a linear regression data mining techniques based on the obtained datasets, generate performance metrics from the YOLO algorithm, and estimate their size and weight within three to four months of their grower stage and four to five months for the finisher stage. The YOLO algorithm, live feed, histogram equalization, and objective detection are a few examples of techniques used for image processing.

Capstone Project! MILKFISH GROWTH CLASSIFICATION SYSTEM USING YOLO ALGORITHM

Background of the study

This study aims to develop a software application that can detect milkfish in live feed frames, estimate their size and weight using the YOLO algorithm, estimate the fish length, width and weight, and display graphs for data visualization.

Significance of the Study

The proposed system has the potential to significantly improve the effectiveness and accuracy of fish farming and fisheries studies, which will save money and enhance sustainable development.
